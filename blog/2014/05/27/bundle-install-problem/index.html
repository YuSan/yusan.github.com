Linux执行bundle Install出现的问题解决

May 27th, 2014 | Comments

今天配置好了系统，然后想把octopress也重新搞好，在执行bundle install时出现了下面的错误

于是上网找解决办法，差不多找到了类似的问题，下面有解决的办法，按照那个说的搞定了，顺便写下来记录一下!

即安装ruby-dev版本，终端执行：

sudo apt-get install ruby1.9.1-dev

问题解决！
